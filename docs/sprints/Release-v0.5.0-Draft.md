# Release Notes: v0.5.0 (Draft)

## 🏷️ Release Tag
`v0.5.0` (Draft for Review)

---

## 🔬 Scientific Additions
* **CPCB Target Reconstruction Integration**: Automatically resolves and recalculates AQI targets from pollutant concentration inputs using official segmented formulas if the overall `AQI` column is missing or empty.
* **Temporal Hold-Out Split Strategy**: Incorporates chronologically ordered train/test splits (rather than random shuffling) when temporal indexes are present, providing a scientifically sound estimate of real-world forecasting drift.
* **Mean Bias Error (MBE)**: Integrates MBE to track systemic model drift and prediction bias (over-prediction vs under-prediction).

---

## ⚙️ Engineering Additions
* **Unified preprocessing Pipelines**: Bundles ColumnTransformers (scaling, imputation, One-Hot Encoding) and the RandomForestRegressor inside a single, self-contained `sklearn.pipeline.Pipeline`.
* **Hyperparameter Configuration Lock**: Pulls all model hyperparameters dynamically from `config.py` instead of hardcoding parameters in execution scripts.
* **Transformed Feature Importances**: Maps Random Forest feature importances directly to post-ColumnTransformer features (mapping dummy classes generated by OHE).
* **Checkpoints & Metadata Serialization**: Exports `.joblib` model binaries and `.json` performance/validation metrics to the designated output folder.

---

## 📊 Validation & Test Summary
* **Unit & Integration Tests**: Added a dedicated test suite verifying predictions, metrics, and serialization.
* **Execution Outcome**: 33/33 tests passed successfully.
* **Coverage Areas**: CPCB formulas, GEE collection filters, Feature Engineering schema bounds, and Random Forest serialization/metrics.

---

## ⚠️ Known Limitations
* Preprocessing imputer defaults to simple column means.
* Collinearity filtering using Pearson correlation assumes linear trends.

---

## 🛠️ Upgrade Notes
* Add `joblib` and `scikit-learn` to the python environment if not already installed.
* When predicting using the saved `baseline_model.joblib`, pass the raw `DataFrame` directly; the preprocessing pipeline handles scaling and One-Hot Encoding internally.
